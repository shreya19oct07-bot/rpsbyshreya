🎮 Rock Paper Scissors Game

A simple and fun Rock Paper Scissors game built in Python while learning programming basics 😄

🚀 Features
- Play against computer 🤖
- Best of 5 rounds system
- Score tracking 📊
- Input validation
- Emoji-based interaction 🎨

🛠️ Tech Used
- Python
- Google Colab

▶️ How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Enter your name
4. Play by typing: "rock", "paper", or "scissors"
